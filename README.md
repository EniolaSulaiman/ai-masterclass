# AI Automation & No-Code Freelancing Masterclass — Sales Page

A fully responsive, high-converting sales page for an AI freelancing course. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, no build tools.


---

## 🔗 Live Demo

[View Live Site](https://eniola-sulaiman.github.io/ai-masterclass/)

---

## 📌 About the Project

This is a portfolio project demonstrating a production-ready course sales page. The page is designed to convert visitors into buyers through structured copywriting, social proof, urgency mechanics, and a clean dark-theme UI.

**Course concept:** *AI Automation & No-Code Freelancing Masterclass — Build $3K/Month Skills in 60 Days*

---

## ✨ Features

- Fully responsive layout — mobile, tablet, and desktop
- Countdown timer with live urgency bar
- SVG icon system — zero emojis, all inline SVGs
- Scroll-triggered fade-up animations via `IntersectionObserver`
- Interactive FAQ accordion with ARIA accessibility
- Two-tier pricing section with "Best Value" ribbon
- Testimonials grid with student earning badges
- Bonus section with value stack
- 30-day guarantee block
- Semantic HTML5 with full SEO setup (meta tags, Open Graph, Twitter Card, JSON-LD schema)

---

## 🛠️ Built With

## ♿ Accessibility

- Semantic landmarks: `<main>`, `<nav>`, `<footer>`, `role="complementary"`
- All sections use `aria-labelledby` pointing to heading IDs
- FAQ toggle updates `aria-expanded` on open/close
- All decorative SVGs marked `aria-hidden="true"`
- Star ratings include `aria-label` text equivalents
- `:focus-visible` outlines on interactive elements

---

## 🔍 SEO

- Keyword-rich `<title>` tag (under 70 characters)
- Meta `description` (under 160 characters)
- `rel="canonical"` link
- Open Graph tags for social sharing
- Twitter Card tags
- JSON-LD structured data: `Course` schema + `FAQPage` schema
